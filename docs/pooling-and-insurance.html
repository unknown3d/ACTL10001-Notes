What is Risk Pooling?

This is a random variable - we do not know what it will be until the losses occur

  • Expected value of the average loss for the pool \[E[A]=\mu_A=E[\frac{\sum_{i=1}^n(X_i)}{\emph{n}}]\] The Number of losses (in this case) has a Binomial (,) distribution, so the probability that individuals incur a loss is \[Pr(\sum_{i=1}^{n} X_i=\emph{jL})= \binom{n}{j} q^j(1-q)^{n-j}, j=0,1,2…,n\]

    Note: \[\binom{n}{k} = \frac{n!}{k!(n-k)!}\] \[Pr(N=j)\]

    N = total number of losses

- Expected Value of the Average Loss \[\mu_A=E[\frac{\sum_{i=1}^n(X_i)}{\emph{n}}]=E[\frac{N \cdot L}{n}]=\frac{\emph{L}}{\emph{n}}E[\emph{N}]\] \[=\frac{\emph{L}}{\emph{n}}\sum_{j=0}^{n} \emph(j) \binom{n}{j}q^j(1-q)^{n-j}\]

\[=\frac{\emph{L}}{\emph{n}} \cdot \emph{nq}=\emph{qL}\]
Expected Value of the average loss in the pool is the same as the expected value of the individual loss

We have \[E[k]= k\]
